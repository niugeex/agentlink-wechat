# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning as a working convention.

## [0.1.0] - 2026-03-28

### Added
- Initial public release of the AgentLink WeChat TypeScript SDK.
- QR login flow, session persistence, long polling, and message event handling.
- Text replies, reply stream support, typing indicator, and markdown normalization.
- Image and file upload/download helpers with AES media handling.
- Multi-account support, allowlist pairing mode, and built-in slash commands.
- Runnable examples including `echo-bot` and the OpenAI-powered local doc assistant demo.
